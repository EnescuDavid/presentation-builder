After the `<!-- END MODE B -->` comment block and before the closing `-->` of Mode C, add a new comment section:

```html
<!-- THEME CSS ================================================================
     Include ONE theme file's CSS in the <style> block below the tokens.
     Available themes:
       - themes/default/theme.css    (Consulting Standard — navy/blue)
       - themes/startup/theme.css    (Bold, vibrant — orange/blue, Space Grotesk)
       - themes/enterprise/theme.css (Conservative — navy/gray, traditional)

     To use a theme: copy its CSS content into the style block below,
     after the animation rules and before the master layer CSS.

     To switch themes: replace the theme CSS block with another theme's content.
     All 14 components adopt the new branding automatically via CSS custom properties.

     For custom themes from PPTX extraction:
       1. Run: node tools/extract-theme.js path/to/template.pptx themes/[name]
       2. Claude reads the extracted JSON and assembles theme CSS
       3. Include the generated theme CSS here
     ========================================================================== -->
```

Insert this comment block right after the Mode C comment section ends, before the `<style>` tag.

Also add a placeholder comment inside the `<style>` tag after the animations section and before the master layer CSS:

