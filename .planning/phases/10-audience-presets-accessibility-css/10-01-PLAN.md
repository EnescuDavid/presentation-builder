**4. Append print CSS block to themes/_base.css** (per D-05, D-06, D-07, FIX-08):

After the audience presets block, append:

```css
/* ==========================================================================
   Print / PDF Styles
   Per D-06: Single @media print block in _base.css shared by all themes
   Per D-07: Master footer kept visible for branding/legal context
   ========================================================================== */

@media print {
  /* Force backgrounds and colors */
  * {
    print-color-adjust: exact !important;
    -webkit-print-color-adjust: exact !important;
  }

  /* One slide per page */
  .reveal .slides section {
    page-break-after: always;
    page-break-inside: avoid;
  }

  /* Hide reveal.js chrome */
  .reveal .controls,
  .reveal .progress,
  .reveal .slide-number {
    display: none !important;
  }

  /* Keep master footer visible (D-07) */
  #masterLayer {
    display: block !important;
    position: fixed;
    bottom: 0;
  }

  /* Show all fragments in print */
  .reveal .fragment {
    opacity: 1 !important;
    visibility: visible !important;
    transform: none !important;
    filter: none !important;
  }

  /* Disable animations in print */
  *, *::before, *::after {
    animation-duration: 0s !important;
    transition-duration: 0s !important;
  }
}
```
