```css
/* ==========================================================================
   Keyboard Focus Indicators (A11Y-06)
   :focus-visible shows focus ring only for keyboard navigation, not mouse clicks
   ========================================================================== */

.reveal .slides a:focus-visible,
.reveal .slides button:focus-visible,
.reveal .slides [tabindex]:focus-visible {
  outline: 3px solid var(--color-accent);
  outline-offset: 2px;
  border-radius: var(--radius-sm);
}

/* Ensure focus is visible on dark backgrounds */
.reveal .slides section[data-background-color="dark"] a:focus-visible,
.reveal .slides section[data-background-color="dark"] button:focus-visible,
.reveal .slides section[data-background-color="dark"] [tabindex]:focus-visible {
  outline-color: var(--color-background);
}
```

Place this block between the dark variant section and the audience presets section in _base.css. If plan 01 has already run, insert before the `/* Audience Presets */` comment. If not, append at end of file.

For each of the 21 component templates, add ARIA attributes to the `<section>` element and to visual sub-elements. Use `role="group"` (not `role="region"` to avoid landmark pollution per research). Use German aria-labels since the framework is German-first (per D-09).

**Simple templates (role + aria-label on section only):**

For each template listed below, add `role="group"` and `aria-label="..."` to the opening `<section>` tag:

1. **title.html**: `role="group" aria-label="Titelfolie"`
2. **section-break.html**: `role="group" aria-label="Abschnittstrenner"`
3. **text-heavy.html**: `role="group" aria-label="Textfolie"`
4. **two-column.html**: `role="group" aria-label="Zwei-Spalten-Folie"`
5. **agenda.html**: `role="group" aria-label="Agenda"`
6. **summary.html**: `role="group" aria-label="Zusammenfassung"`
7. **comparison.html**: `role="group" aria-label="Vergleich"`
8. **card-grid.html**: `role="group" aria-label="Kartengitter"`
9. **quote.html**: `role="group" aria-label="Zitat"`
10. **contact.html**: `role="group" aria-label="Kontakt"`
11. **timeline.html**: `role="group" aria-label="Zeitstrahl"`
12. **code-block.html**: `role="group" aria-label="Codebeispiel"`
13. **data-table.html**: `role="group" aria-label="Datentabelle"` (also verify the table uses semantic `<table>` element; if it does, no additional table role needed)

**Visual components (role + aria-label + alt text slots):**

14. **metrics.html**: `role="group" aria-label="Kennzahlen"` on section. Add `aria-hidden="true"` to any sparkline/micro-chart decorative SVGs or CSS shapes within metric cards (they are visual decoration of the number, not standalone content).

15. **image-full-bleed.html**: `role="group" aria-label="Vollbild" aria-roledescription="Bildfolie"` on section. Since the image is a `data-background-image` (creates a div, not an img), the section's `aria-label` serves as the alt text. Add a comment in the template: `<!-- Replace aria-label with descriptive text for the background image -->`.

16. **chart.html**: `role="group" aria-label="Diagramm"` on section. On the `<canvas>` element, add `role="img" aria-label="Diagrammbeschreibung hier einf&uuml;gen"`. Add a comment: `<!-- Replace canvas aria-label with chart description, e.g. "Balkendiagramm zeigt Umsatz 2023-2026" -->`.

17. **waterfall.html**: `role="group" aria-label="Wasserfalldiagramm"` on section. On the `<canvas>` element, add `role="img" aria-label="Wasserfalldiagramm-Beschreibung hier einf&uuml;gen"`. Add comment similar to chart.html.

18. **mermaid-diagram.html**: `role="group" aria-label="Diagramm"` on section. On the `.comp-mermaid__diagram` container div (which holds the pre-rendered SVG), add `role="img" aria-label="Diagrammbeschreibung hier einf&uuml;gen"`. Add comment: `<!-- Replace aria-label with diagram description -->`.

19. **framework.html**: `role="group" aria-label="Framework-Matrix"` on section. On the grid container element (`.comp-framework__grid` or equivalent), add `aria-label="2x2-Matrix"`.

20. **harvey-balls.html**: `role="group" aria-label="Bewertungsmatrix"` on section. On each Harvey ball circle element, add `role="img" aria-label="Bewertung: X%"` where X matches the fill percentage (0, 25, 50, 75, or 100). Example: the 75% ball gets `role="img" aria-label="Bewertung: 75%"`.

21. **team.html**: `role="group" aria-label="Team"` on section. On each `<img>` element for team member photos, ensure `alt` attribute exists. If placeholder, use `alt="Foto von [Name]"`. Add comment: `<!-- Replace alt with actual team member name -->`.
