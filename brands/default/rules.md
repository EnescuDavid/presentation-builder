# Brand Rules: Default

## Mandatory Slides
- Title slide with clear headline and subtitle
- Agenda slide for presentations over 8 slides

## Tone & Voice
- Balanced formality -- neither overly casual nor stiff
- Action titles preferred: verb-first German phrasing
- Bullet points should be concise, not narrative paragraphs

## Visual Style
- Clean whitespace, no visual clutter
- Moderate animation density -- entrance animations on key elements only
- All 21 components available without restriction

## Chart & Data Conventions
- Use --color-accent for primary data series
- Label axes in German
- Source citations on data-heavy slides

## Prohibited Patterns
- No restrictions -- the default brand is intentionally permissive

## Corporate Compliance
- No mandatory confidentiality labels
- No mandatory logo placement
